Matplotlib Python library

Matplotlib is a powerful plotting libray for python programming language. It also provides an object-oriented API for embedding plots into applicaiton using general purpose GUI toolskits such TKinter, wxPython, Qt or GTK+

Installation and setting up

To use Matplotlib library, we need to have python installed. If you dont have python installed, download it and install from the official website. (This tutorial is based on Python 3.0+) Next we need to install few packages. To install the packages we are going to use Python Package-management-System (pip). We need to install Matplotlib, Numpy and Pandas. To install them type the following command in your terminal:

root@snapf ~ # pip install matplotlib numpy pandas
